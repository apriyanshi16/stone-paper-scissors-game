# Stone Paper Scissors Game

A web-based **Stone Paper Scissors** game developed using **HTML, CSS, and JavaScript**, focusing on clean UI, logical game flow, and interactive user experience.

---

## Key Highlights
- Player vs Computer gameplay
- Implemented core game logic using JavaScript
- Dynamic result display (Win / Lose / Draw)
- Responsive and user-friendly interface
- Structured and readable codebase

---

## Tech Stack
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
